# GM命令工具
CommandManager for Unity.

主要功能
* 生成/解析 xml 格式的命名列表 参考 CommandManager->SaveCommands 以及 CommandManager->LoadCommands
* 简易用户界面 命令列表 显示开关/选择/输入/提交
* 简易用户界面 Trace 日志 显示开关
* 命令提交事件机制 CommandManager->onSubmit
